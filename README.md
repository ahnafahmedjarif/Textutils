## Run Locally
To run this project you need to install python on your system.

Clone the project

```bash
  git clone https://github.com/ahnafahmedjarif/Textutils.git
```

Go to the project directory

```bash
  cd Textutils
```

Install dependencies

```bash
  pip3 install django
```

Start the server

```bash
  python3 manage.py runserver
```
